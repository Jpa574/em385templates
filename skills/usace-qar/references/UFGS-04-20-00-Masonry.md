# UFGS 04 20 00 - Masonry

## Key Spec Highlights for QAR

### Materials
- **Brick:** ASTM C216 (face brick), C62 (building brick)
- **Concrete masonry units (CMU):** ASTM C90
- **Mortar:** ASTM C270 (M, S, N types)
- **Grout:** ASTM C476
- **Reinforcement:** ASTM A615

### Critical Inspection Points

1. **Mortar & Grout**
   - Mortar type per spec (M, S, N)
   - Grout consistency (flow)
   - No additives unless spec'd

2. **Masonry Units
   - Layout (running bond, stack bond)
   - Unit quality (no cracks, chips)
   - Wet units pre-wetted in hot weather

3. **Installation**
   - Joints properly filled (head joints first)
   - Joint tooling (properly compacted)
   - Reinforcement in place (ladder, truss)
   - Anchors at interfaces

4. **Curing**
   - Protect from sun and rain initially
   - Keep moist for curing period

### Documentation Required
- Shop drawings (masonry layout, details)
- Product data (brick, CMU, mortar)
- Grout test reports
- Prism tests (if required)

### Common Deficiencies
- Wrong mortar type
- Joints not fully filled
- Missing reinforcement
- No control joints
- Units not properly cured
- Cleanup inadequate (staining)
