# Roofing Inspection Procedures (DFOW)

## Roofing Defined Feature of Work

### Phase 1: Pre-Con (Before Roofing)

**Contractor Must Complete:**
- [ ] Roofing submittals approved (membrane, insulation, flashing)
- [ ] Roof insulation R-value per spec
- [ ] Membrane type per spec
- [ ] Warranty requirements defined
- [ ] AHA for roofing approved
- [ ] Weather restrictions reviewed
- [ ] Deck inspection complete

**QAR Verification:**
- [ ] Verify insulation R-value
- [ ] Confirm membrane type
- [ ] Check deck condition

**Forms/Docs Required:**
- Roofing Submittal Log
- Deck Inspection Report

---

### Phase 2: Work in Progress (During Roofing)

**Contractor Must Complete:**
- [ ] Deck clean, dry, free of debris
- [ ] Insulation installed per spec
- [ ] Cover board installed (if required)
- [ ] Membrane installed correctly
- [ ] Seaming per manufacturer
- [ ] Flashing at perimeters, penetrations
- [ ] Edge details per spec
- [ ] Walk pads installed (if required)

**QAR Verification:**
- [ ] Verify insulation thickness
- [ ] Check membrane laps
- [ ] Verify flashing details
- [ ] Check edge securement

**Testing:**
| Test | When | Who Performs | Who Witnesses |
|------|------|--------------|---------------|
| Flood Test | Low-slope areas | Contractor | QAR |
| Visual Inspection | All areas | Contractor | QAR |

---

### Phase 3: Closeout (Final Inspection)

**Contractor Must Complete:**
- [ ] All seams inspected
- [ ] Flashings complete
- [ ] No leaks (verified by flood test or visual)
- [ ] Warranty documentation
- [ ] O&M manuals
- [ ] As-builts marked
- [ ] Roof accessories installed (hatches, curbs)

**QAR Verification:**
- [ ] Verify flood test passed
- [ ] Check all flashings
- [ ] Verify no ponding water
- [ ] Confirm warranty

**Forms/Docs Required:**
- Flood Test Report
- Roofing Final Sign-off

## Common Roofing Deficiencies

| Deficiency | Spec Reference | Common Cause |
|------------|----------------|--------------|
| Membrane not sealed | Manufacturer | Improper seaming |
| Wrong insulation R-value | Spec | Substitution |
| Ponding water | Spec | Inadequate slope |
| Missing flashing | Spec | Missed detail |
| Membrane damage | Spec | Foot traffic, debris |
| Wrong membrane type | Spec | Substitution |
