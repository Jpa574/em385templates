# Phase 2: Submittals

Submittals happen after Design Review, during Pre-Con and into early construction. Contractor submits shop drawings, product data, and material certs. QAR reviews for spec compliance.

## Submittal Types

| Type | What It Is | QAR Review |
|------|-----------|------------|
| **Shop Drawings** | Custom-fabricated items (steel, concrete, ductwork) | Matches specs/drawings |
| **Product Data** | Manufacturer specs, cut sheets, certifications | Materials meet spec |
| **Material Certificates** | Mill certs, test reports, compliance docs | Per UFGS requirements |
| **Design Data** | Engineering calcs, structural calcs | Meets spec requirements |
| **O&M Manuals** | Operation & maintenance docs | For closeout |

## Submittal Review Process

### 1. Log Submittal
- Assign submittal number (e.g., SD-01, SD-02)
- Note spec section requiring it
- Set schedule deadline

### 2. Review Against Spec
- Check materials meet UFGS requirements
- Verify dimensions match drawings
- Flag any deviations

### 3. Stamp Decision

```
Government Review Stamp options:
- APPROVED
- APPROVED AS NOTED (minor comments, contractor proceeds)
- DISAPPROVED (significant deviations, must resubmit)
- DEFERRED SUBMITTAL (not required now, later)
```

### 4. Return with Comments
- List all comments on transmittal
- Cite spec section for each comment
- Return within timeframe specified in contract

## Review Checklist by Trade

### Fire Suppression (UFGS 21 13 00)
- [ ] Sprinkler heads: temperature rating, type (pendent, upright, etc.)
- [ ] Pipe: material matches spec (carbon steel,.schedule)
- [ ] Fittings: grooved vs. threaded, correct type
- [ ] Valves: UL/FM listed
- [ ] Alarm devices: waterflow, pressure switch

### HVAC (UFGS 23 05 15)
- [ ] Pipe material: matches spec (BCS, copper, etc.)
- [ ] Fittings: correct type for pipe material
- [ ] Insulation: thickness, material per spec
- [ ] Heat tracing: self-regulating, IEEE 515
- [ ] Valves: correct type, pressure class

### Electrical (UFGS 26 05 00)
- [ ] Conduit: type matches location (wet/dry)
- [ ] Wire: insulation type (THWN/THHN)
- [ ] Devices: hospital-grade or hard-use
- [ ] Panelboards: bolt-on breakers
- [ ] Transformers: NEMA ST 20 testing

---

## Common Submittal Issues

1. **Wrong material** — Contractor submitts PVC where spec requires steel
2. **Missing certifications** — No UL/FM listing provided
3. **Incomplete data** — Missing manufacturer, model number
4. **Dimension mismatch** — Equipment doesn't fit space
5. **Spec revision outdated** — Using old spec version
