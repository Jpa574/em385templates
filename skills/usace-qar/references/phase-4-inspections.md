# Phase 4: Inspections

Inspections happen throughout construction. QAR observes work, documents findings, and ensures compliance with specs and approved submittals.

## Types of Inspections

### 1. Pre-Cover Inspection
Before work is concealed (concrete pour, behind drywall, underground piping)
- Verify existing conditions
- Check installed materials match approved submittals
- Document installation per spec

### 2. In-Progress Inspection
During work to catch issues early
- Verify work matches specs/drawings
- Check installation per manufacturer instructions
- Observe workmanship quality

### 3. Final Inspection
When work is complete
- Verify all punch items closed
- Confirm systems function (start-up, testing)
- Review O&M manuals received

---

## What QAR Inspects

### Always Check:
- **Materials** — Are approved materials being used? (match submittals)
- **Installation** — Per spec? Per manufacturer?
- **Workmanship** — Industry standard quality?
- **Testing** — Are required tests performed? Results documented?

### Key Inspection Points by Trade

#### Fire Suppression
- Pipe hangers per MSS SP-58 (spacing, type)
- Joints properly made (welded, grooved per spec)
- Sprinkler orientation correct
- Thrust blocks at bends/tees
- Alarm valve installed per manufacturer

#### HVAC
- Pipe hangers per spec (spacing)
- Insulation complete and correct thickness
- Heat trace properly installed
- Ductwork supports per SMACNA
- Equipment set per manufacturer's requirements

#### Electrical
- Conduit supports per spec
- Box fill calculations
- Wire gauge correct
- Connections proper (solder, crimp per spec)
- Grounding continuous

---

## Documentation During Inspection

### Daily Report (Daily QC Report)
- Date, weather, work force
- Work performed by area/trade
- inspections performed
- Tests witnessed
- Issues found / deficiencies noted
- Contractor notified of items

### Deficiency Report
- Location (building, floor, area)
- Description of issue
- Spec/drawing reference
- Required corrective action
- Photos (if applicable)

---

## Inspection vs. Testing

**QAR Inspects** — Workmanship, materials, installation
**Testing Lab Tests** — Concrete strength, weld X-ray, electrical certification

QAR witnesses testing but doesn't perform it.

---

## Closeout of Inspection Phase

- All inspections documented
- Pre-cover inspections signed off
- Testing results reviewed and accepted
- Punch list generated (if applicable)
