# Concrete Inspection Procedures (DFOW)

## Concrete Defined Feature of Work

### Phase 1: Pre-Con (Before Concrete Placement)

**Contractor Must Complete:**
- [ ] Concrete mix design submitted (60 days before)
- [ ] Mix design approved by QAR
- [ ] Submittals approved (aggregates, cement, admixtures)
- [ ] QC Plan includes concrete work
- [ ] AHA for concrete placement approved
- [ ] Testing lab contracted, certified
- [ ] Weather protection plan (hot/cold weather)
- [ ] Curing plan submitted
- [ ] Formwork inspection completed
- [ ] Reinforcement inspection completed
- [ ] Embeds, sleeves in place, verified

**QAR Verification:**
- [ ] Review mix design against spec
- [ ] Verify mill certs for cement, aggregates
- [ ] Confirm reinforcement per drawings
- [ ] Check cover thickness
- [ ] Verify formwork per spec
- [ ] Witness formwork inspection

**Forms/Docs Required:**
- Mix Design Approval Form
- Formwork Inspection Report
- Reinforcement Inspection Report
- Concrete Placement AHA

---

### Phase 2: Work in Progress (During Placement)

**Contractor Must Complete:**
- [ ] Weather within limits (per spec)
- [ ] Concrete temperature verified
- [ ] Slump test performed
- [ ] Air content tested (if required)
- [ ] Cylinders cast per ASTM C31
- [ ] Placement method per spec
- [ ] Consolidation (vibration) complete
- [ ] Cold joints prevented
- [ ] curing initiated on time
- [ ] Protection from damage

**QAR Verification:**
- [ ] Witness slump test
- [ ] Witness cylinder cast
- [ ] Document batch plant, truck numbers
- [ ] Record weather conditions
- [ ] Monitor placement rate
- [ ] Verify curing method
- [ ] Document start/end time

**Testing Requirements:**
| Test | Frequency | Who Performs | Who Witnesses |
|------|-----------|--------------|---------------|
| Slump | Each batch | Testing Lab | QAR |
| Air Content | Each batch | Testing Lab | QAR |
| Temperature | Each batch | Testing Lab | QAR |
| Cylinders | Per spec (min 2 sets) | Testing Lab | QAR |

**Forms/Docs Required:**
- Daily Concrete Placement Log
- Slump Test Report
- Cylinder Cast Log
- Curing Record

---

### Phase 3: Closeout (After Cure/Removal)

**Contractor Must Complete:**
- [ ] Cylinder break results meet spec (f'c)
- [ ] Surface finish per spec
- [ ] Floor flatness verified
- [ ] Core tests (if required)
- [ ] Repairs complete
- [ ] All formwork removed

**QAR Verification:**
- [ ] Review cylinder break results
- [ ] Verify 28-day strength achieved
- [ ] Inspect finished surface
- [ ] Check floor tolerances (F-number system)
- [ ] Verify repairs acceptable

**Forms/Docs Required:**
- Concrete Strength Test Report
- Floor Flatness Report
- Final Concrete Inspection Sign-off

## Common Concrete Deficiencies

| Deficiency | Spec Reference | Common Cause |
|------------|----------------|--------------|
| Low Strength | ACI 318, Spec f'c | Improper cure, high w/c |
| Honeycomb | UFGS 03 30 00 | Incomplete consolidation |
| Cold Joints | UFGS 03 30 00 | Placement delay |
| Cracking | UFGS 03 30 00 | Improper curing, shrinkage |
| Out-of-Tolerance | Drawings | Formwork deflection |
| Incorrect Cover | UFGS 03 30 00 | Spacer failure |
