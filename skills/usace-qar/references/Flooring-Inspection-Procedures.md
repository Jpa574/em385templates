# Flooring Inspection Procedures (DFOW)

## Flooring Defined Feature of Work

### Phase 1: Pre-Con (Before Installation)

**Contractor Must Complete:**
- [ ] Flooring submittals approved (tile, carpet, VCT, LVT)
- [ ] Adhesive per spec
- [ ] Subfloor moisture testing complete
- [ ] Concrete cure time verified (28 days min for tile)
- [ ] Floor levelness verified
- [ ] Environment conditions adequate
- [ ] AHA for flooring approved

**QAR Verification:**
- [ ] Verify submittals approved
- [ ] Check moisture test results
- [ ] Verify subfloor prep

**Testing:**
| Test | Requirement | Who Performs | Who Witnesses |
|------|-------------|--------------|---------------|
| Moisture (CH) | <3 lbs/1000 SF/24hr | Testing Lab | QAR |
| Moisture (RH) | <75% RH | Testing Lab | QAR |

---

### Phase 2: Work in Progress (During Installation)

**Contractor Must Complete:**
- [ ] Adhesive spread per manufacturer
- [ ] Temperature within limits
- [ ] Joint spacing per spec
- [ ] Pattern alignment
- [ ] Transitions at doorways
- [ ] Expansion joints maintained

**QAR Verification:**
- [ ] Check adhesive coverage
- [ ] Verify joint spacing
- [ ] Check pattern alignment

---

### Phase 3: Closeout (Final Inspection)

**Contractor Must Complete:**
- [ ] Flooring adhered properly
- [ ] No delamination
- [ ] Transitions complete
- [ ] Elevation changes <1/4" (ADA)
- [ ] No lippage
- [ ] Cleaning complete
- [ ] O&M manuals
- [ ] Maintenance data

**QAR Verification:**
- [ ] Verify adhesion (no hollow sounds)
- [ ] Check lippage <1/4"
- [ ] Verify transitions
- [ ] Confirm floor clean

**Forms/Docs Required:**
- Moisture Test Report
- Flooring Final Inspection Sign-off

## Common Flooring Deficiencies

| Deficiency | Spec Reference | Common Cause |
|------------|----------------|--------------|
| Delamination | Manufacturer | Moisture, adhesive failure |
| Lippage >1/4" | ADA | Poor installation |
| Moisture damage | Spec | Inadequate testing |
| Joint separation | Spec | Wrong adhesive |
| Pattern misalignment | Spec | Poor layout |
| Staining | Spec | Improper cleanup |
