# Doors, Frames & Hardware Inspection Procedures (DFOW)

## Doors Defined Feature of Work

### Phase 1: Pre-Con (Before Installation)

**Contractor Must Complete:**
- [ ] Door submittals approved (wood, steel, hollow metal)
- [ ] Frame submittals approved
- [ ] Hardware schedule approved
- [ ] Keying schedule approved (if required)
- [ ] Fire-rated door labels verified
- [ ] ADA-compliant hardware specified
- [ ] AHA for door installation approved

**QAR Verification:**
- [ ] Verify door types match schedule
- [ ] Confirm fire ratings where required
- [ ] Verify hardware grades

**Forms/Docs Required:**
- Door Schedule Approval
- Hardware Schedule Approval

---

### Phase 2: Work in Progress (During Installation)

**Contractor Must Complete:**
- [ ] Frames plumb, level, secure
- [ ] Doors sized correctly
- [ ] Door operation (swing, latch, close)
- [ ] Hardware installed per schedule
- [ ] Hinges correct type, properly attached
- [ ] Locksets operational
- [ ] Closers adjusted per spec
- [ ] Panic hardware where required

**QAR Verification:**
- [ ] Check frame installation
- [ ] Verify door operation
- [ ] Check hardware function
- [ ] Verify ADA heights (hardware 34"-48")

**Pre-Cover Inspection:**
- [ ] All frames installed
- [ ] All doors hung
- [ ] Hardware functional

---

### Phase 3: Closeout (Final Inspection)

**Contractor Must Complete:**
- [ ] All doors operational
- [ ] All hardware adjusted
- [ ] All locksets working
- [ ] All closers adjusted
- [ ] Fire-rated doors labeled
- [ ] Keys provided (if applicable)
- [ ] O&M manuals
- [ ] As-builts marked

**QAR Verification:**
- [ ] Verify door operation
- [ ] Check hardware function
- [ ] Verify fire labels present
- [ ] Check ADA compliance

**Forms/Docs Required:**
- Door Final Inspection Sign-off

---

## Doors Inspection Checklists

### Door Pre-Cover Inspection
```
DOORS PRE-COVER INSPECTION

Project: _________________ Date: __________
Location: Building ___ Floor: _________

FRAMES:
[ ] Frames plumb
[ ] Frames level
[ ] Frames secure to wall
[ ] Frames weatherstripped (exterior)

DOORS:
[ ] Door size correct
[ ] Door fits frame (clearances per spec)
[ ] Door operation smooth

HARDWARE:
[ ] Hinges installed (3 min)
[ ] Hardware per schedule

QAR SIGN-OFF: _________________ DATE: __________
```

---

### Door Final Inspection
```
DOORS FINAL INSPECTION

Project: _________________ Date: __________
Building: _________ Area: _________

OPERATION:
[ ] Door swings freely
[ ] Latch engages properly
[ ] Door closes completely
[ ] No binding or sticking

HARDWARE:
[ ] Hinges operational, no squeaks
[ ] Locksets function properly
[ ] Closers adjusted (proper speed)
[ ] Panic hardware operational (fire doors)
[ ] ADA heights correct (34"-48")

FIRE RATED:
[ ] Labels intact and visible
[ ] Fire-rated assemblies identified

DOCUMENTATION:
[ ] Keying schedule provided
[ ] O&M manuals received

QAR ACCEPTANCE: _________________ DATE: __________
```

## Common Door Deficiencies

| Deficiency | Spec Reference | Common Cause |
|------------|----------------|--------------|
| Door not operational | Spec | Installation error |
| Wrong clearance | Spec | Frame out of square |
| Missing fire label | IBC | Wrong door used |
| Hardware not per schedule | Spec | Substitution |
| ADA height wrong | ADA | Installation error |
| Closer improper adjustment | Spec | Missed adjustment |
