# Structural Steel Inspection Procedures (DFOW)

## Structural Steel Defined Feature of Work

### Phase 1: Pre-Con (Before Erection)

**Contractor Must Complete:**
- [ ] Shop drawings approved
- [ ] Mill certifications submitted
- [ ] Steel grades per spec (A992, A36, etc.)
- [ ] Bolt specifications approved (A325, A490)
- [ ] Weld procedure specifications (WPS) approved
- [ ] Welder certifications current
- [ ] Erection drawings approved
- [ ] AHA for steel erection approved
- [ ] Foundation anchors in place
- [ ] Steel staged, inspected upon arrival

**QAR Verification:**
- [ ] Verify shop drawings match specs
- [ ] Confirm mill certs match steel delivered
- [ ] Check bolt grades match spec
- [ ] Verify WPS approved
- [ ] Verify welder certs current

**Forms/Docs Required:**
- Shop Drawing Approval Form
- Mill Certification Log
- Welder Qualification List
- Bolt Testing Report

---

### Phase 2: Work in Progress (During Erection)

**Contractor Must Complete:**
- [ ] Field connections made up
- [ ] High-strength bolts properly tensioned
- [ ] Welds completed per WPS
- [ ] Temporary bracing in place
- [ ] Alignment within tolerances
- [ ] Field bolts snugged before final tensioning
- [ ] Direct tension indicator (DTI) used where required

**QAR Verification:**
- [ ] Witness bolt installation
- [ ] Verify bolt tensioning procedure
- [ ] Check alignment per AISC
- [ ] Witness start of field welding
- [ ] Verify weld quality (visual)

**Testing Requirements:**
| Test | When | Who Performs | Who Witnesses |
|------|------|--------------|---------------|
| Bolt Tension | Each connection | Contractor | QAR |
| DTI Verification | Per spec | Contractor | QAR |
| Visual Weld Inspection | All welds | CWI | QAR |
| NDT (UT/MT/PT) | Per spec | Testing Lab | QAR |

**Pre-Cover Inspection:**
- [ ] All connections complete
- [ ] All bolts tensioned
- [ ] All welds complete
- [ ] Temporary bracing still in place (or removed properly)

---

### Phase 3: Closeout (Final Inspection)

**Contractor Must Complete:**
- [ ] All bolts tensioned, verified
- [ ] All welds complete, NDT done
- [ ] Touch-up primer applied
- [ ] Final alignment verified
- [ ] As-built drawings marked
- [ ] O&M manuals for mechanical finishes

**QAR Verification:**
- [ ] Review bolt test reports
- [ ] Review NDT reports
- [ ] Verify alignment final
- [ ] Check touch-up paint
- [ ] Confirm as-builts accurate

**Forms/Docs Required:**
- Bolt Tension Verification Report
- Weld NDT Report
- Steel Final Inspection Sign-off

## Common Structural Steel Deficiencies

| Deficiency | Spec Reference | Common Cause |
|------------|----------------|--------------|
| Wrong bolt grade | AISC, Spec | Substitution |
| Bolts not tensioned | AISC | Improper installation |
| Missing bridge rails | Spec | Oversight |
| Weld defects | AWS D1.1 | Improper welding |
| Wrong steel grade | Spec | Substitution |
| Misalignment | AISC | Erection error |
| No fireproofing | Spec | Missed area |
