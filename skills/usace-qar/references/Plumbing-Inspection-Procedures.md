# Plumbing Inspection Procedures (DFOW)

## Plumbing Defined Feature of Work

### Phase 1: Pre-Con (Before Rough-In)

**Contractor Must Complete:**
- [ ] Plumbing submittals approved (pipe, fittings, valves, fixtures)
- [ ] Pipe material per spec
- [ ] Fixture schedule approved
- [ ] Water heater specs approved (if required)
- [ ] Backflow preventer specs approved
- [ ] AHA for plumbing approved
- [ ] Materials on site verified

**QAR Verification:**
- [ ] Verify pipe material per application
- [ ] Confirm fixtures match schedule
- [ ] Verify backflow prevention

**Forms/Docs Required:**
- Plumbing Submittal Log
- Fixture Schedule Approval

---

### Phase 2: Work in Progress (During Rough-In)

**Contractor Must Complete:**
- [ ] Underground piping installed per spec
- [ ] Drainage slope correct (1/4" per foot min)
- [ ] Vent piping correct
- [ ] Water supply piping correct
- [ ] Pipe supports per MSS SP-58
- [ ] Valves accessible
- [ ] Cleanouts at required locations
- [ ] Backflow preventers installed

**QAR Verification:**
- [ ] Check drainage slope
- [ ] Verify vent connections
- [ ] Check pipe supports
- [ ] Verify cleanout locations

**Testing:**
| Test | When | Who Performs | Who Witnesses |
|------|------|--------------|---------------|
| Drainage Hydrostatic | Before cover-up | Contractor | QAR |
| Water Supply Pressure | Before cover-up | Contractor | QAR |
| Gas Test | If gas present | Contractor | QAR |

**Pre-Cover Inspection:**
- [ ] All underground piping complete
- [ ] Drainage tested
- [ ] Water supply tested
- [ ] All connections accessible

---

### Phase 3: Closeout (Final Inspection)

**Contractor Must Complete:**
- [ ] All fixtures installed
- [ ] All faucets operational
- [ ] All drains functional
- [ ] Water heater installed, started (if applicable)
- [ ] Backflow preventers certified
- [ ] O&M manuals (2 copies)
- [ ] As-builts marked up

**QAR Verification:**
- [ ] Verify all fixtures operational
- [ ] Check drain flow
- [ ] Verify backflow cert
- [ ] Confirm O&M receipt

**Forms/Docs Required:**
- Plumbing Pressure Test Report
- Backflow Certification
- Final Plumbing Sign-off

## Common Plumbing Deficiencies

| Deficiency | Spec Reference | Common Cause |
|------------|----------------|--------------|
| Drainage slope wrong | IPC, Spec | Improper installation |
| No air gap | Spec | Code violation |
| Missing cleanouts | IPC | Missed locations |
| Wrong pipe material | Spec | Substitution |
| No backflow prevention | IPC | Missed device |
| Valve not accessible | Spec | Missed location |
| Leaks at joints | Spec | Improper installation |
