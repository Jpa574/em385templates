# UFGS 07 52 00 - Modified Bituminous Membrane Roofing

## Key Spec Highlights for QAR

### Materials
- **Membrane:** Modified bitumen (SBS, APP)
- **Base sheet:** Per spec type
- **Flashing:** Modified bitumen or metal
- **Insulation:** Per spec (XPS, polyiso)
- **Cover board:** Per spec

### Critical Inspection Points

1. **Substrate**
   - Roof deck condition (steel, concrete, wood)
   - Insulation installed properly
   - Deck secure, no deflection

2. **Membrane Installation**
   - Proper adhesion (torch, self-adhering, cold adhesive)
   - Side laps and end laps per manufacturer
   - Seaming (hot air weld, adhesive)
   - Flashing at perimeters and penetrations

3. **Testing**
   - Flood test (if low-slope)
   - Visual inspection for leaks
   - Core samples

### Documentation Required
- Shop drawings (roof plan, details)
- Product data (membrane, insulation)
- Manufacturer certifications
- Test reports (flood test, core cuts)

### Common Deficiencies
- Wrong membrane type for slope
- Improper lap spacing
- Missing flashing at edges/penetrations
- Membrane damage (puncture, blister)
- Insulation not adhered
- Ponding water (inadequate slope)
