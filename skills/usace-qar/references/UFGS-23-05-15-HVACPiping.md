# UFGS 23 05 15 - Common Piping for HVAC

## Key Spec Highlights for QAR

### Materials
- **Black Carbon Steel (BCS):** Schedule 40, ASTM A53 (seamless or ERW)
- **Galvanized Carbon Steel (GCS):** For drain/waste/vent
- **Copper:** ASTM B88 (seamless water tube)
- **Fittings:** Malleable iron screwed (ASTM A197) for DN50+, steel butt weld (ASTM A234) for larger
- **Heat Tracing:** Self-regulating, IEEE 515, UL listed

### Critical Inspection Points
1. **Materials**
   - Verify pipe schedule matches spec (Schedule 40 for most)
   - Check fitting types match pipe size/thickness

2. **Installation**
   - Welded connections wherever possible
   - Flexible metal hose for steam (vertical preferred)
   - Access for maintenance - check access doors

3. **Testing**
   - Hydrostatic test
   - Air test
   - Heat trace cable test (continuity + insulation resistance)
   - System operation test

4. **Electrical**
   - Motors meet NEMA MG-11 efficiency
   - Phase protection for motors >1HP
   - Max 120V control circuits

### Documentation Required
- Shop Drawings
- Product Data (all pipe, fittings, valves)
- Test Reports (hydrostatic, heat trace)
- O&M Manuals

### Common Deficiencies
- Wrong pipe schedule (e.g., using Schedule 10 instead of 40)
- Missing access doors for concealed work
- Heat trace not properly secured
- No insulation on underground copper
- Missing hangers/supports per MSS SP-58
