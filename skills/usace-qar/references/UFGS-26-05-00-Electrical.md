# UFGS 26 05 00.00 40 - Common Work Results for Electrical

## Key Spec Highlights for QAR

### Conductors
- **Min Sizes:** Branch circuits = #12 AWG, Class 1 signal = #14 AWG
- ** Insulation:** THWN/THHN (600V), XHHW, or RHW
- **Age Limit:** No cable manufactured >12 months before delivery

### Conduit Types
- **Rigid Steel:** Hot-dipped galvanized, NEMA RN 1, ANSI C80.1, UL 6
- **EMT:** Zinc-coated steel, UL 797. Must use raintight compression fittings (crimp/setscrew NOT allowed)
- **Flexible:** Galvanized steel, UL 1
- **Liquidtight:** PVC over galvanized steel core
- **PVC:** Schedule 40 minimum, NEMA TC 2

### Critical Inspection Points

1. **Conduit Installation**
   - Rigid steel: threaded fittings, solid gaskets on blank covers
   - EMT: compression-type with insulated throat only
   - Underground: per Section 3.2.7.6

2. **Panelboards**
   - Bolt-on breakers only (plug-in prohibited)
   - 10,000A minimum interruption capacity
   - Main buses independent of breakers

3. **Receptacles**
   - Hard use/hospital grade only (no residential)
   - Weatherproof = UL wet-location listed + weatherproof-while-in-use cover

4. **Safety Switches**
   - Cannot open cover when switch is ON
   - Coinproof cover release
   - Lockable in OFF position

### Testing Required
- Continuity test
- Phase-rotation test
- Insulation resistance test (Megger)
- 600V wiring test
- Ground-fault receptacle test

### Documentation Required
- Shop Drawings (marking strips)
- Product Data (conduit, wire, panels, breakers, transformers)
- Test Reports (all tests above)
- Transformer factory tests (NEMA ST 20)

### Common Deficiencies
- Wrong conduit type for environment (e.g., EMT in wet location)
- EMT using setscrew/crimp fittings instead of compression
- Residential-grade receptacles in commercial/MILCON
- Missing gaskets on conduit fittings in wet areas
- Panelboard with plug-in breakers
- No weatherproof covers on outdoor outlets
