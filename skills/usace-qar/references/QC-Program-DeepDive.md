# QC Program Deep Dive

## UFGS 01 45 00 - Quality Control

This is the core spec for contractor QC. QAR must know what's required.

---

## QC Plan Requirements

### Required Submittals (QCPM submits):
1. **QC Plan** — Within 30 days of NTP
   - Organization chart
   - QC Manager qualifications
   - Inspection procedures
   - Testing procedures
   - Nonconformance procedures
   - Pre-cover inspection procedures

2. **QC Manager Qualifications**
   - USACE/NAVAF certification
   - Experience on similar projects
   - Full-time commitment

3. **Three-Day Notice**
   - QCPM notifies QAR 3 days before:
     - Tests
     - Critical inspections
     - Concrete pours
     - Steel erecti

---

## Three-Phase Control - Detailed

### PHASE 1: Pre-Construction (Before Work Starts)

**QCPM must complete and document:**

| Item | QCPM Action | QAR Verification |
|------|-------------|------------------|
| QC Manager assigned | Name submitted | On-site? |
| QC Plan approved | Copy to QAR | Reviewed? |
| Submittals received | 100% of spec items | All received? |
| Materials approved | Verified in submittals | Approved? |
| Mockups complete | Per spec | Acceptable? |
| Testing laboratory | Approved, certified | Scheduled? |

**No work begins until Phase 1 complete.**

---

### PHASE 2: Work in Progress

**Daily Requirements:**

1. **QC Inspection Report** - Each day
   - Areas inspected
   - Work observed
   - Tests performed/witnessed
   - Nonconformance items
   - Safety observations

2. **Three-Day Lookahead**
   - Work planned for next 3 days
   - Tests scheduled
   - Inspections needed
   - Critical items

**Weekly Requirements:**

1. **Weekly QC Report** to QAR
   - Work completed
   - Status of submittals
   - Status of RFIs
   - Deficiency log status
   - Testing status
   - Photos

**Monthly Requirements:**

1. **Monthly QC Report**
   - Overall project status
   - Schedule status
   - Manpower by trade
   - QA/QC issues
   - Upcoming milestones

---

### PHASE 3: Pre-Final and Final

**Pre-Final Inspection:**
- QCPM conducts walk-through
- Generates punch list
- Contractor corrects
- QAR verifies closure

**Final Inspection:**
- All punch items closed
- All testing complete
- O&M manuals received
- As-builts submitted
- Government acceptance

## Testing Requirements

### QCPM Coordinates:

| Test Type | Who Performs | Who Witnesses |
|-----------|--------------|---------------|
| Concrete cylinders | QCPM or testing lab | QAR witnesses |
| Steel inspection | Testing lab | QAR witnesses |
| compaction | Testing lab | QAR witnesses |
| Electrical | Licensed tester | QAR witnesses |
| Pressure tests | QCPM | QAR witnesses |

### QAR Role:
- **Witness** tests (don't perform)
- **Verify** test results meet spec
- **Document** in daily report
- **Request** retest if failed

---

## Deficiency Management (QCPM Side)

### QCPM Process:

1. **Issue NCR** (Nonconformance Report)
2. **Assign** to contractor for correction
3. **Verify** correction
4. **Close** NCR

### QAR Process:

1. **Review** contractor's NCRs
2. **Verify** closure
3. **Issue** government deficiency if needed
4. **Track** separately from contractor NCRs

**Note:** Government and contractor deficiency tracking are separate systems.

|----------|-------------|

| Daily inspections done? | QC Inspection Report |
| Tests scheduled? | Three-day lookahead |
| Tests witnessed? | Daily report |

| Weekly reports submitted? | Transmittal log |
| Phase 1 complete? | Checklist |

---

## Red Flags QAR Catches

| Warning Sign | What It Means |
|--------------|---------------|
| No QC Inspection Report | QCPM not working |
| Tests not witnessed | Testing may not be done |
| Deficiencies not closed | Quality issues piling up |
| No Three-Day Lookahead | Poor planning |
| QC Manager not on site | Contractor in violation |
| Weekly reports missing | QC program failing |
| Punch items reopened | Poor workmanship |

---

## Key Distinction

**QCPM responsibilities:**
- Inspect all work
- Document all inspections
- Coordinate testing
- Track deficiencies
- Report to government

**QAR responsibilities:**
- Verify QC doing their job
- Witness testing
- Government-side inspection (separate)
- Document concerns

**Separate but complementary.**
