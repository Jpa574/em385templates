# UFGS 07 21 00 - Thermal Insulation

## Key Spec Highlights for QAR

### Materials
- **Board insulation:** XPS, EPS, Polyiso per spec
- **Blanket insulation:** Fiberglass, mineral wool
- **Spray foam:** Closed-cell, open-cell per application

### Critical Inspection Points

1. **Installation**
   - Correct R-value per spec
   - Proper thickness
   - Complete coverage (no gaps)
   - Vapor barrier where required
   - Insulation appropriate for location (interior, exterior, roof)

2. **Mechanical**
   - Pipe insulation: proper thickness per pipe size
   - Duct insulation: FSK faced, sealed joints
   - Equipment insulation: removable blankets where needed

3. **Fire Rating**
   - Flame spread index
   - Smoke developed index
   - Firestopping at penetrations

### Documentation Required
- Product data (R-values, fire ratings)
- Thickness schedule
- Test reports (thermal performance)

### Common Deficiencies
- Wrong R-value installed
- Gaps in insulation
- Missing vapor barrier
- Pipe insulation not continuous
- Duct insulation not sealed
- Firestopping missing at penetrations
