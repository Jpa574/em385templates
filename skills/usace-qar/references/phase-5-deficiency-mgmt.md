# Phase 5: Deficiency Management

Deficiency management runs throughout construction. QAR identifies issues, documents them, tracks correction, and verifies closure.

## Deficiency Categories

### Minor
- Workmanship cosmetic issues
- Does not affect function or code
- Correctable quickly
- e.g., paint touch-up, minor hardware adjustment

### Major
- Affects function but easily corrected
- Requires rework but no structural impact
- e.g., drywall finish, valve adjustment

### Critical
- Code violation
- Safety hazard
- Structural impact
- Requires immediate stop of work in some cases
- e.g., missing fire stopping, improper grounding

---

## Documentation Format

### Deficiency Notice Elements:
```
DEFICIENCY NOTICE #: [DN-001]
DATE: [Date]
PROJECT: [Project Name/Contract #]
LOCATION: [Building, Floor, Area, Room #]
DISCIPLINE: [Trade - Electrical, Mechanical, etc.]

DEFICIENCY:
[Description of what's wrong]

SPEC/DRAWING REFERENCE:
[Spes section, drawing number, detail]

REQUIRED CORRECTION:
[What needs to be done to fix]

DUE DATE: [Date for correction]
RESPONSE REQUIRED BY: [Date]
```

---

## Tracking

### Deficiency Log Tracks:
- DN number
- Date issued
- Location
- Description
- Category (Minor/Major/Critical)
- Due date
- Response date
- Status (Open / Corrected / Verified Closed)
- Photos

---

## Resolution Workflow

```
1. ISSUE deficiency
        ↓
2. CONTRACTOR acknowledges
        ↓
3. CONTRACTOR corrects issue
        ↓
4. QAR verifies correction
        ↓
5. CLOSE deficiency (with sign-off)
```

---

## Common Deficiencies

### Fire Suppression
- Missing thrust blocks
- Wrong sprinkler type for location
- Missing hangers/supports
- No fire stopping at penetrations
- Alarm components not wired

### HVAC
- Missing insulation
- Ductwork not sealed
- Heat trace not functional
- Equipment not anchored
- Filter type wrong

### Electrical
- Conduit not supported
- Missing grounding
- Wire gauge undersized
- Boxes not accessible
- Panelboard not labeled

---

## QAR Role in Deficiency Mgmt

1. **Document clearly** — Specific location, issue, spec cite
2. **Set realistic due dates** — Consider scope of correction
3. **Follow up** — Don't let deficiencies linger
4. **Verify properly** — Actually inspect the fix, don't just take contractor's word
5. **Close completely** — Sign-off only after verified
