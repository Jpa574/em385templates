# UFGS 14 21 00 - Elevators

## Key Spec Highlights for QAR

### Materials
- **Elevator type:** Hydraulic, traction, machine-room-less
- **Speed/Capacity:** Per spec
- **Cab finish:** Per spec

### Critical Inspection Points

1. **Installation
   - Guide rails alignment
   - Hydraulic cylinder/cable condition
   - Platform/shaft dimensions

2. **Safety
   - Safety device tests (overspeed, buffer)
   - Door operators (photocell, safety edge)
   - Emergency lighting/communication

3. **Testing
   - Load test (125% capacity)
   - Safety device tests
   - Fire service operation
   - ADA operation (accessibility)

### Documentation Required
- Shop drawings
- Product data
- Test certificates
- O&M manuals
- Fire service documentation

### Common Deficiencies
- Wrong elevator type for building
- Safety devices not tested
- Door timing incorrect
- ADA compliance issues
- Fire service not functional
- Documentation incomplete
