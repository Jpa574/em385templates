# Electrical Inspection Procedures (DFOW)

## Electrical Defined Feature of Work

### Phase 1: Pre-Con (Before Rough-In)

**Contractor Must Complete:**
- [ ] Electrical submittals approved (conduit, wire, panels, devices)
- [ ] Panel schedules approved
- [ ] Load calculations approved (if required)
- [ ] Conduit fill calculations (if required)
- [ ] Wire gauge/sizing per spec
- [ ] AHA for electrical work approved
- [ ] Materials on site, verified against submittals
- [ ] Conduit routing reviewed
- [ ] Panel locations verified
- [ ] Coordination with other trades (MEP)

**QAR Verification:**
- [ ] Confirm all submittals approved
- [ ] Verify materials match approved submittals
- [ ] Check panel locations per drawings
- [ ] Verify coordination clearances

**Forms/Docs Required:**
- Electrical Submittal Log
- Material Verification Form
- Panel Schedule Approval

---

### Phase 2: Work in Progress (During Rough-In)

**Contractor Must Complete:**
- [ ] Conduit installed per spec (type, size, support)
- [ ] Support spacing per MSS SP-58
- [ ] Boxes installed at correct heights
- [ ] Grounding continuous
- [ ] Conduit properly connected
- [ ] Box fill per NEC
- [ ] Conduit protected where required

**QAR Verification:**
- [ ] Verify conduit type matches location (wet/dry)
- [ ] Check support spacing (max 10' for EMT)
- [ ] Verify box locations per plan
- [ ] Check grounding connections
- [ ] Verify proper box fill
- [ ] Witness conductor installation

**Testing Requirements:**
| Test | When | Who Performs | Who Witnesses |
|------|------|--------------|---------------|
| Continuity | Before close-up | Contractor | QAR |
| Insulation Resistance | Before close-up | Contractor | QAR |
| Ground Continuity | Before close-up | Contractor | QAR |

**Pre-Cover Inspection (Before Walls Closed):**
- [ ] All conduit installed, bonded
- [ ] All boxes in place, secured
- [ ] All wiring complete
- [ ] All grounds continuous
- [ ] No damage to conductors
- [ ] Junction boxes accessible

---

### Phase 3: Closeout (Final Inspection)

**Contractor Must Complete:**
- [ ] All circuits labeled
- [ ] Panel directories complete
- [ ] All devices installed (receptacles, switches)
- [ ] All fixtures installed
- [ ] GFCI tested operational
- [ ] Circuit testing complete
- [ ] Load test performed
- [ ] O&M manuals for equipment
- [ ] As-built drawings marked up

**QAR Verification:**
- [ ] Verify all labeling complete
- [ ] Test GFCI receptacles
- [ ] Verify circuit operation
- [ ] Check panel directory accuracy
- [ ] Witness equipment start-up
- [ ] Verify O&M manual receipt

**Testing Requirements:**
| Test | When | Who Performs | Who Witnesses |
|------|------|--------------|---------------|
| Continuity | Final | Contractor | QAR |
| Insulation Resistance | Final | Contractor | QAR |
| Megger Test 600V | Final | Testing Lab | QAR |
| Ground Fault Test | Final | Contractor | QAR |
| Load Test | Final | Contractor | QAR |

**Forms/Docs Required:**
- Electrical Pre-Cover Inspection Form
- Insulation Resistance Test Report
- Panel Directory
- Circuit Directory
- Final Electrical Inspection Sign-off

## Electrical Inspection Checklists (Standardized)

### Pre-Cover Electrical Inspection
```
ELECTRICAL PRE-COVER INSPECTION

Project: _________________ Date: __________
Location: Building ___ Floor: _________ Room: _________
Contractor: _________________

CONDUIT INSPECTION:
[ ] Conduit type correct (EMT/RMC/IMC)
[ ] Supports per MSS SP-58 (max 10' spacing)
[ ] Conduit properly connected (wet locations)
[ ] Conduit protected from damage
[ ] Conduit fill within code

BOXES INSPECTION:
[ ] Boxes secure, mounted plumb
[ ] Box fill within code limits
[ ] Device plates will be flush
[ ] Box height per ADA (receptacles 15"-48")
[ ] Accessible (no blocked access)

WIRING INSPECTION:
[ ] All conductors continuous
[ ] No splices in concealed locations
[ ] Conductor gauge correct per circuit
[ ] Wire nuts secure, insulated
[ ] Grounding conductors complete

GROUNDING:
[ ] Equipment grounds continuous
[ ] Ground bonds at panels
[ ] Ground at all devices

JUNCTION BOXES:
[ ] All junction boxes accessible
[ ] Boxes covered (no open boxes)
[ ] Junction boxes with blank covers identified

QAR SIGN-OFF: _________________ DATE: __________
```

## Common Electrical Deficiencies

| Deficiency | Spec Reference | Common Cause |
|------------|----------------|--------------|
| EMT using setscrew fittings | UFGS 26 05 00 | Wrong fitting type |
| Conduit supports too far apart | MSS SP-58 | >10' spacing |
| Panel with plug-in breakers | UFGS 26 05 00 | Wrong breaker type |
| No GFCI in wet locations | NEC | Missed location |
| Wire gauge undersized | NEC, Spec | Wrong wire size |
| Missing box supports | Spec | Improper installation |
| Residential grade devices | Spec | Wrong device grade |
| Unlabeled circuits | Spec | Incomplete labeling |
