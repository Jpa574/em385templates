# Phase 6: Project Closeout

Closeout happens at the end of construction. QAR verifies all work is complete, all punch items resolved, and all documentation is submitted.

## Closeout Activities

### 1. Substantial Completion
- Contractor declares substantial completion
- QAR walks project with contractor
- Generate **punch list** of items to complete
- Note deficiencies for correction

### 2. Correction Period
- Contractor corrects punch items
- QAR verifies each closure
- May be multiple punch lists (list #1, #2, etc.)

### 3. Final Completion
- All punch items closed
- All tests completed and documented
- All O&M manuals received and reviewed
- All training completed (if required)
- Government accepts final completion

---

## Punch List Format

```
PUNCH LIST # [1]
PROJECT: [Project Name/Contract #]
DATE: [Walkthrough Date]

| # | Location | Item | Category | Due Date | Closed | Date Closed |

| 2 | Electrical Room | Panel label missing | Minor | -- | -- | -- |
| 3 | Mechanical Room | VAV not functioning | Major | -- | -- | -- |
```

---

## Closeout Documentation

### Required Submittals:
- [ ] O&M Manuals (per spec, typically 2 copies)
- [ ] As-Built Drawings (marked-up record drawings)
- [ ] Test Reports (all acceptance testing)
- [ ] Material Certificates (closeout documentation)
- [ ] Warranty Documentation
- [ ] Training Records (if equipment requires)
- [ ] Startup Forms (for mechanical/electrical systems)

### QAR Verification:
- [ ] All deficiencies closed
- [ ] All tests passed
- [ ] All documentation received
- [ ] All systems functioning
- [ ] All training complete

---

## Common Closeout Issues

1. **Missing O&M manuals** — Contractor delays
2. **Punch list items reopened** — Workmanship fails after weather cycles
3. **Incomplete as-builts** — Contractor didn't mark changes
4. **Training not scheduled** — Equipment but no training
5. **Warranty docs missing** — Long-term liability gaps

---

## Contractor's Obligation

Contractor must complete ALL work in contract SOW, including:
- All punch items from substantial completion
- All required testing
- All documentation submittals
- All training

Government is not obligated to accept until ALL is complete.

---

## End of Project

After final acceptance:
- File all records
- Release lien waivers (if applicable)
- Final payment processed

Project complete.
