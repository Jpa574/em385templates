# Drywall/Gypsum Board Inspection Procedures (DFOW)

## Drywall Defined Feature of Work

### Phase 1: Pre-Con (Before Installation)

**Contractor Must Complete:**
- [ ] Drywall submittals approved (type, gauge)
- [ ] Board type per location (regular, Type X, moisture-resistant)
- [ ] Framing inspection complete
- [ ] Substrate conditions verified
- [ ] Environment conditions adequate (temp/humidity)
- [ ] AHA for drywall approved

**QAR Verification:**
- [ ] Verify board type per location
- [ ] Check framing for board attachment
- [ ] Verify environment acceptable

**Forms/Docs Required:**
- Drywall Submittal Log

---

### Phase 2: Work in Progress (During Installation)

**Contractor Must Complete:**
- [ ] Board properly attached
- [ ] Spacing per ASTM C840 (12" OC max for walls, 16" for ceilings)
- [ ] Joints per spec (tape, mud, finish level)
- [ ] Fastener type per spec (screws, nails)
- [ ] Fire-rated assemblies per design
- [ ] Moisture-resistant board in wet areas

**QAR Verification:**
- [ ] Check fastener spacing
- [ ] Verify board type per area
- [ ] Check fire-rated assemblies
- [ ] Verify finish level

**Pre-Cover Inspection:**
- [ ] All board installed
- [ ] All fasteners in place
- [ ] Fire assemblies noted

---

### Phase 3: Closeout (Final Inspection)

**Contractor Must Complete:**
- [ ] Finish level per spec (0-5)
- [ ] No surface defects
- [ ] All joints taped, mudded
- [ ] All penetrations fire-stopped
- [ ] Paint ready (no priming required)
- [ ] O&M manuals (if applicable)

**QAR Verification:**
- [ ] Check finish quality
- [ ] Verify surface defects repaired
- [ ] Check fire-stopping at penetrations

**Forms/Docs Required:**
- Drywall Final Inspection Sign-off

## Drywall Inspection Checklists

### Drywall Pre-Cover Inspection
```
DRYWALL PRE-COVER INSPECTION

Project: _________________ Date: __________
Location: Building ___ Floor: _________

FRAMING:
[ ] Stud spacing correct (12"/16" OC)
[ ] Blocking in place
[ ] Framing meets deflection criteria

BOARD:
[ ] Type correct (Regular/Type X/Moisture-resistant)
[ ] Gauge correct (1/2" or 5/8")
[ ] Fasteners correct (screws preferred)

INSTALLATION:
[ ] Fastener spacing (12" OC walls, 16" OC ceilings)
[ ] Fastener depth (1/4" below surface)
[ ] Joints staggered
[ ] Fire-rated assemblies noted

```

## Common Drywall Deficiencies

| Deficiency | Spec Reference | Common Cause |
|------------|----------------|--------------|
| Wrong board type | Spec | Missed location |
| Fastener spacing too far | ASTM C840 | Improper installation |
| Fastener protruding | Spec | Over-driven |
| Cracking at joints | Spec | Movement, poor tape |
| No fire-stopping | Spec | Missed penetrations |
| Finish level wrong | Spec | Missed specification |
