# Inspection Procedures - Organized by DFOW

## Three Phases of Construction (Per UFGS 01 45 00)

| Phase | Timing | QAR Role | Contractor Role |
|-------|--------|----------|-----------------|
| **Phase 1: Pre-Con** | Before work starts | Verify ready | Complete prep |
| **Phase 2: Work in Progress** | During construction | Witness/verify | Inspect continuously |
| **Phase 3: Closeout** | Work complete | Final inspection | Complete punch items |

---

## DFOW Inspection Matrix

| DFOW | Phase 1 (Pre-Con) | Phase 2 (In-Progress) | Phase 3 (Closeout) |
|------|-------------------|----------------------|-------------------|
| Concrete Work | Submittals, mix design, forms ready | Placement, curing, testing | Finish, cure verification |
| Structural Steel | Shop drawings, bolting specs | Erection, connections | Final inspection, touch-up |
| Electrical | Submittals, layout review | Rough-in, above ceiling | Final, testing, labeling |
| HVAC | Submittals, equipment location | Ductwork, piping, insulation | Start-up, balancing |
| Plumbing | Submittals, rough-in | Piping, fixtures | Final, pressure test |
| Fire Protection | Submittals, sprinkler layout | Rough-in, hydrotest | Final flow test |
| Doors/Windows | Submittals, hardware schedule | Installation | Final adjustment |
| Roofing | Submittals, weatherproofing | Membrane, flashing | Final inspection, leak test |

## Standardized Forms Needed

### Phase 1: Pre-Con Checklist
- [ ] Submittals approved
- [ ] Materials on site
- [ ] QC Plan approved
- [ ] AHA approved
- [ ] Mockups complete (if required)
- [ ] Testing lab scheduled
- [ ] Pre-construction conference held

### Phase 2: In-Progress Inspection Form
- [ ] Work matches approved submittals
- [ ] Installation per spec
- [ ] Testing performed
- [ ] Documentation complete
- [ ] Deficiencies identified
- [ ] Corrected and closed

### Phase 3: Closeout Checklist  
- [ ] All punch items closed
- [ ] Testing complete
- [ ] O&M manuals received
- [ ] As-builts submitted
- [ ] Training complete
- [ ] Final inspection passed
