# UFGS 09 90 00 - Painting and Coatings

## Key Spec Highlights for QAR

### Materials
- **Paints:** Generic type per spec (alkyd, latex, epoxy)
- **Systems:** Prime + intermediate + finish
- **VOC limits:** Per project location (LEED, local regs)

### Critical Inspection Points

1. **Surface Preparation**
   - Cleanliness (SSPC standards)
   - Profile for primer adhesion
   - Moisture content in wood

2. **Application
   - Proper thickness (DFT per coat)
   - Proper cure time between coats
   - Coverage (no misses, holidays)

3. **Finished Product
   - Color match
   - sheen uniformity
   - No runs, sags, orange peel

### Documentation Required
- Product data (all paints, primers)
- Color schedule
- Dry film thickness reports

### Common Deficiencies
- Wrong paint system
- Insufficient surface prep
- Thickness below spec
- Wrong thinner used
- Not cured between coats
- Color mismatch
