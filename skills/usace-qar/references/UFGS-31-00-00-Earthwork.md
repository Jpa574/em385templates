# UFGS 31 00 00 - Earthwork

## Key Spec Highlights for QAR

### Materials
- **Fill:** Select fill, general fill, rock fill per spec
- **Backfill:** Native or imported
- **Geotextiles:** Per application

### Critical Inspection Points

1. **Excavation
   - Bottom condition (no soft spots)
   - Grade correct
   - Benches/terraces if slope

2. **Fill Placement
   - Moisture content (optimum ±2%)
   - Lift thickness (6" loose max, compacted per spec)
   - Compaction (proctor density)

3. **Testing
   - In-place density tests (nuclear gauge or sand cone)
   - Moisture tests
   - Bearing capacity verification

### Documentation Required
- Shop drawings (grading plan)
- Test reports (compaction, proctor)
- Borrow source approval

### Common Deficiencies
- Poor compaction
- Wrong fill material
- Moisture out of range
- Lift thickness exceeds limit
- No testing documentation
- Soft spots not removed
