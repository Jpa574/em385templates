# UFGS 09 60 00 - Flooring

## Key Spec Highlights for QAR

### Materials
- **Tile:** Ceramic, porcelain, quarry tile per spec
- **Carpet:** ASTM E648, carpet tile or broadloom
- **Resilient:** VCT, LVT, rubber, sheet vinyl
- **Concrete:** Polished, stained, overlayment

### Critical Inspection Points

1. **Substrate Preparation**
   - Concrete cure time (28 days min for tile)
   - Moisture content (calcium chloride test)
   - Levelness tolerance
   - Clean, no contaminants

2. **Installation
   - Adhesive type per flooring
   - Joint spacing (grout joints for tile)
   - Pattern alignment
   - Expansion joints

3. **Finished Floor
   - Elevation transitions
   - ADA compliance (no lippage >1/4")
   - Color variation within tolerance

### Documentation Required
- Shop drawings (pattern, layout)
- Product data (all flooring materials)
- Moisture test reports
- Maintenance data

### Common Deficiencies
- Moisture issues (delamination)
- Wrong adhesive
- Lippage exceeding tolerance
- Staining or discoloration
- Expansion joints missing
- Pattern not aligned
