# UFGS 28 31 70 - Fire Alarm System, Addressable

## Key Spec Highlights for QAR

### System Requirements
- **Addressable system:** Each device has unique address
- **Control panel:** UL listed, NFPA 72 compliant
- **Monitoring:** Fire alarm monitoring company required

### Inspection Points

1. **Devices**
   - Smoke detectors: correct type for location
   - Heat detectors: fixed temp vs rate-of-rise
   - Pull stations: accessible, correct height
   - notification appliances: strobes, horns, speakers

2. **Wiring**
   - Circuit wiring per NFPA 70
   - Wire gauge per voltage drop calculations
   - Conduit or cable type per spec

3. **Testing**
   - Functional test: all devices
   - Flow switch verification
   - Pull station alarm verification
   - Notification appliance audibility/visibility

### Documentation Required
- Shop drawings (device locations, wiring)
- Device address list
- Battery calculations
- Testing certifications
- NFPA 72 certification

### Common Deficiencies
- Detector spacing exceeds code
- Pull station height wrong
- Strobe intensity inadequate
- Wire gauge undersized
- Missing supervision
- Battery backup inadequate
