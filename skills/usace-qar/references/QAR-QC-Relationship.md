# QAR & QC Manager Relationship

## Who Does What

### Contractor QC Manager (QCPM)
**Responsible for:**
- Contractor Quality Control program
- Daily, Weekly, Monthly QC reports
- Three-phase QC control (UFGS 01 45 00)
- All inspections (not QAR)
- Testing coordination
- Documentation

### QAR (Government)
**Responsible for:**
- Verify contractor's QC program is working
- Witness testing (don't perform it)
- Review QC documentation
- Government inspections separate from contractor
- Documentation of QC performance

---

## Three-Phase QC Control (Per UFGS 01 45 00)

### Phase 1: Project Start Up (Work Starts)
- QC Manager assigned
- QC Plan submitted and approved
- All submittals received
- Materials approved
- Mockups complete
- Pre-construction conferences

### Phase 2: Work in Progress
- Daily QC inspections by contractor
- Weekly QC reports (by QCPM)
- Monthly QC reports
- Testing per spec
- Deficiency tracking and closure
- Weekly / monthly Sonderberg audits

### Phase 3: Work Complete
- Pre-final inspection
- Punch list development and closure
- Final inspection
- O&M manuals received
- As-built drawings
- All testing documentation

---

## QC Reports QAR Reviews

### Daily QC Report (From Contractor)
- Work performed that day
- QC inspections conducted
- Tests performed/witnessed
- Quality observations
- Safety observations
- Material deliveries
- Workforce count by trade

### Weekly QC Report (QCPM Sign-off)
- Contractor's weekly activities
- Status of submittals
- Status of shop drawings
- RFI log status
- Deficiency log status
- Upcoming work
- Testing schedule
- Three-week lookahead

### Monthly QC Report
- Project status overall
- Schedule status
- Cost status
- Manpower
- QA/QC issues
- Design issues
- Status of all submittals

---

## QAR Verification of QC

QAR verifies QCPM is doing their job:

| QCPM Should | QAR Verifies |
|-------------|--------------|
| Conduct daily inspections | Review daily QC report |
| Document findings | Check for comprehensive log |
| Issue deficiencies | Verify closure |
| Witness testing | Witness same tests |
| Submit weekly reports | Review for completeness |
| Maintain submittal log | Verify tracking current |

---

## QAR Does NOT Replace QC

**QAR does NOT:**
- Inspect instead of contractor
- Create contractor's documentation
- Perform testing
- Manage contractor's schedule

**QCPM does NOT:**
- Represent government
- Accept work for government
- Stop work on government's behalf (can for safety)

---

## Interaction Flow

```
Morning: QAR reviews Daily QC Report
                ↓
Day: QAR observes contractor work + testing
                ↓
After: QAR documents observations
                ↓
Contractor: Addresses any issues in next Daily QC Report
                ↓
QAR: Verifies in next review cycle
```

---

## Key Point

**Contractor inspects. QAR verifies contractor inspected.**

The QCPM is the contractor's quality control. The QAR is the government's quality assurance. Different roles, different responsibilities.

QAR catches what QC misses — that's the point of the check-and-balance.
