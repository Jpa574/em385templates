# UFGS 08 11 00 - Doors, Frames, and Hardware

## Key Spec Highlights for QAR

### Materials
- **Doors:** Wood, steel, aluminum, hollow metal per spec
- **Frames:** Wood, steel, aluminum, knock-down vs welded
- **Hardware:** Per builder's hardware schedule

### Critical Inspection Points

1. **Door Installation**
   - Frame plumb and level
   - Door operation (swing, latch, close)
   - Clearances per spec (bottom, sides)
   - ADA compliance (hardware height)

2. **Hardware**
   - Hinges: 3 per door minimum, proper type
   - Locksets: function per schedule
   - Closers: proper adjustment, LCN per spec
   - Panic hardware: correct type, accessible

3. **Finishes**
   - Door finish per spec (paint, stain, overlay)
   - Frame finish matching
   - Hardware finish matching (US26D, etc.)

### Documentation Required
- Shop drawings (door schedules, hardware)
- Product data (doors, frames, hardware)
- Keying schedule
- Hardware templates

### Common Deficiencies
- Wrong door type (fire-rated required)
- Missing fire-rated labels
- Hardware not per schedule
- Door not operating properly
- Wrong clearances
- ADA hardware height
