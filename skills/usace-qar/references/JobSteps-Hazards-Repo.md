# Job Steps & Hazards Repository

## Logical Sequence for AHA Development

### Step 1: Identify the Work Activity
- Break work into discrete activities
- Each activity = one AHA

### Step 2: List ALL Associated Tasks
- Every task within that activity
- Include setup and cleanup

### Step 3: Identify Hazards for EACH Task
- What can go wrong?
- What can hurt workers?

### 4: Assess Risk (RAC)
- Severity x Probability = RAC
- Rate each hazard

### 5: Develop Controls
- Use hierarchy to reduce RAC
- Eliminate → Substitute → Engineer → Admin → PPE

## Common Work Activities - Repository

### 1. Electrical Rough-In

**Associated Tasks:**
- Conduit bending (hand/bender)
- Conduit installation (above ceiling, underground)
- Box mounting
- Panel installation
- Wire pulling
- Grounding connections
- Splicing/junction boxes
- Testing/continuity checks
- Material handling
- Ladder/scaffold work

**Typical Hazards:**

| Task | Hazards | Typical Controls |
|------|---------|------------------|
| Conduit bending | Cuts, ergonomic, noise | Gloves, machine guards |
| Conduit installation | Falls, falling objects, electrical | PFAS, hard hat, LOTO |
| Box mounting | Falling objects, strains | Toe boards, team lift |
| Wire pulling | Electrical, ergonomic | GFCI, rope/dollies |
| Grounding | Electrical | LOTO, test before touch |
| Ladder work | Falls | PFAS, 3-point contact |

**Common RAC Issues:**
- Working near energized conductors (I-II)
- Falls from elevation (I-II)
- Strains from pulling wire (III-IV)

## Control Selection Guide

### To Reduce FALLS (I-II):
| Control Type | Example |
|--------------|---------|
| Elimination | Do work from ground |
| Substitution | Use pre-fab panels |
| Engineering | Guardrails, scaffolds |
| Administrative | Training, flagmen |
| PPE | PFAS, safety nets |

### To Reduce STRUCK-BY (I-II):
| Control Type | Example |
|--------------|---------|
| Elimination | Remove overhead work |
| Substitution | Use mechanical handling |
| Engineering | Barriers, guards |
| Administrative | Traffic control, spotters |
| PPE | Hard hat, high-vis |

### To Reduce CAUGHT-IN (I-II):
| Control Type | Example |
|--------------|---------|
| Elimination | No trench entry |
| Substitution | Pipe sections vs. sections |
| Engineering | Trench boxes |
| Administrative | Training, competent person |
| PPE | Tight-fit clothing |

### To Reduce ELECTRICAL (I-II):
| Control Type | Example |
|--------------|---------|
| Elimination | De-energize |
| Substitution | Battery tools |
| Engineering | Insulation, GFCI |
| Administrative | LOTO, live-work permits |
| PPE | Insulated tools, arc flash |

### To Reduce CHEMICAL/HAZMAT (II-IV):
| Control Type | Example |
|--------------|---------|
| Elimination | Non-toxic substitute |
| Substitution | Less hazardous |
| Engineering | Ventilation, enclosures |
| Administrative | SDS training, PPE |
| PPE | Respirators, gloves |

1. Start with work activity
2. Pull associated tasks from repository (add specifics)
3. Identify hazards from repository (add unique ones)
4. Apply controls from hierarchy
5. Verify RAC reduced to III or lower
