# HVAC Inspection Procedures (DFOW)

## HVAC Defined Feature of Work

### Phase 1: Pre-Con (Before Installation)

**Contractor Must Complete:**
- [ ] HVAC submittals approved (equipment, ductwork, piping, insulation)
- [ ] Equipment submittals approved
- [ ] Ductwork shop drawings approved
- [ ] Piping system design approved
- [ ] Insulation specifications approved
- [ ] AHU/FCU schedules approved
- [ ] Balancing requirements defined
- [ ] Testing requirements defined
- [ ] AHA for HVAC work approved
- [ ] Equipment staged, location reviewed
- [ ] Coordination with electrical, plumbing

**QAR Verification:**
- [ ] Verify all submittals approved
- [ ] Check equipment fit in space
- [ ] Verify duct routing doesn't conflict
- [ ] Check pipe sizing against spec

**Forms/Docs Required:**
- HVAC Submittal Log
- Equipment Layout Approval
- Ductwork Shop Drawing Approval

---

### Phase 2: Work in Progress (During Installation)

#### Ductwork Inspection
**Contractor Must Complete:**
- [ ] Ductwork installed per SMACNA
- [ ] Gauge per spec (pressure class)
- [ ] Support spacing per SMACNA
- [ ] Sealing per spec (Tape/Air seal)
- [ ] Insulation installed
- [ ] Access doors at required locations

**QAR Verification:**
- [ ] Verify gauge matches design
- [ ] Check support spacing
- [ ] Verify seal quality
- [ ] Check insulation thickness
- [ ] Verify access doors

#### Piping Inspection
**Contractor Must Complete:**
- [ ] Pipe material per spec
- [ ] Support spacing per MSS SP-58
- [ ] Insulation complete
- [ ] Test pressures passed
- [ ] Valves accessible

**QAR Verification:**
- [ ] Verify pipe material
- [ ] Check support spacing
- [ ] Witness pressure test
- [ ] Verify insulation

#### Equipment Inspection
**Contractor Must Complete:**
- [ ] Equipment set level
- [ ] Connections complete
- [ ] Filters installed
- [ ] Drives aligned
- [ ] Controls wired

**QAR Verification:**
- [ ] Verify installation per manufacturer
- [ ] Check electrical connections
- [ ] Verify gas/flue connections
- [ ] Check condensate drainage

**Testing Requirements:**
| Test | When | Who Performs | Who Witnesses |
|------|------|--------------|---------------|
| Duct Pressure | Before close-up | Contractor | QAR |
| Piping Hydrostatic | Before close-up | Contractor | QAR |
| Piping Air Test | Before close-up | Contractor | QAR |
| Equipment Start-up | After install | Contractor | QAR |

**Pre-Cover Inspection:**
- [ ] All ductwork installed, sealed
- [ ] All piping tested
- [ ] Insulation complete
- [ ] Access doors in place

---

### Phase 3: Closeout (Final Inspection)

**Contractor Must Complete:**
- [ ] All equipment started
- [ ] Systems balanced
- [ ] Filters replaced (clean)
- [ ] O&M manuals (2 copies)
- [ ] As-builts marked up
- [ ] Training complete (if required)
- [ ] Test and balance report
- [ ] Performance verification

**QAR Verification:**
- [ ] Verify all equipment operational
- [ ] Review T&B report
- [ ] Verify system meets design
- [ ] Check O&M manual receipt
- [ ] Confirm training completed
- [ ] Verify warranty documentation

**Testing Requirements:**
| Test | When | Who Performs | Who Witnesses |
|------|------|--------------|---------------|
| Equipment Start-up | Final | Contractor | QAR |
| System Function Test | Final | Contractor | QAR |
| Air Balance | Final | Contractor | QAR |
| Water Balance | Final | Contractor | QAR |

**Forms/Docs Required:**
- HVAC Pre-Cover Inspection Form
- Pressure Test Report
- Equipment Start-up Form
- Test and Balance Report
- Final HVAC Inspection Sign-off

## HVAC Inspection Checklists (Standardized)

### HVAC Pre-Cover Inspection
```
HVAC PRE-COVER INSPECTION

Project: _________________ Date: __________
Location: Building ___ Floor: _________

DUCTWORK:
[ ] Gauge per design (___ ga)
[ ] Sealed (Tape/Air Seal)
[ ] Supports per SMACNA (max ___" OC)
[ ] Insulation type, thickness correct
[ ] Access doors installed
[ ] Volume dampers installed

PIPING:
[ ] Material per spec
[ ] Supports per MSS SP-58
[ ] Insulation complete, thickness correct
[ ] Test pressure held (___ psi for ___ min)
[ ] Valves labeled, accessible

EQUIPMENT:
[ ] Equipment set level
[ ] Connections complete
[ ] Filters clean, installed
[ ] Drives aligned
[ ] Controls wired

QAR SIGN-OFF: _________________ DATE: __________
```

## Common HVAC Deficiencies

| Deficiency | Spec Reference | Common Cause |
|------------|----------------|--------------|
| Duct not sealed | SMACNA, Spec | Air loss |
| Support spacing wrong | MSS SP-58 | Sagging, failure |
| Missing insulation | Spec | Energy loss |
| Pipe not supported | MSS SP-58 | Vibration, failure |
| No access doors | Spec | Maintenance blocked |
| Equipment not level | Manufacturer | Operation issues |
| Filters wrong size | Spec | Air flow blocked |
| Improper condensate drain | Spec | Water damage |
