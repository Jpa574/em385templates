# Activity Hazard Analysis (AHA) - QAR Review

## What is an AHA

An Activity Hazard Analysis (AHA) is the contractor's safety plan for each distinct work activity. It identifies hazards and controls BEFORE work starts.

**Per USACE EM 385-1-1:** AHA required for all major work activities.

---

## AHA Format Requirements

Each AHA must have:

| Section | Required Content |
|---------|------------------|
| Activity Name | Specific task (e.g., "Electrical Rough-In") |
| Location | Building, floor, area |
| Duration | Start/end dates |
|Supervision | Competent person for this activity |
| Crew Size | How many workers |
| Associated Tasks | ALL tasks within this activity |
| Hazards | Each hazard with RAC score |
| Controls | For EACH hazard |
| PPE | Required for this activity |
| Emergency | Emergency procedures |

---

## Associated Tasks - Critical

**For electrical rough-in, AHA MUST include:**

- Conduit bending and installation
- Wire pulling
- Box installation
- Panel setup
- Grounding
- Splicing
- Testing
- Cleanup
- Material handling
- Ladder work (if elevated)
- Working near other trades

**If tasks are missing → Kick back**

## Risk Assessment Code (RAC)

### What is RAC

| RAC Level | Risk | Action Required |
|-----------|------|-----------------|
| **I** | Critical | Stop work, immediate control |
| **II** | High | Control before next shift |
| **III** | Medium | Control before end of week |
| **IV** | Low | Control as scheduled |

### Control Hierarchy (To Reduce RAC)

1. **ElimINATION** - Remove hazard entirely
2. **Substitution** - Use safer alternative
3. **Engineering** - Isolate workers from hazard
4. **Administrative** - Change work procedures
5. **PPE** - Last line of defense

**Goal:** Reduce RAC to acceptable level (III or lower)

---

## QAR Review in RMS

### System: RMS (Quality Management System)

QAR logs into RMS and:
1. Reviews submitted AHA
2. Checks for completeness
3. Verifies all tasks covered
4. Assesses RAC reduction measures
5. Approves OR kicks back

---

## Kick Back Reasons

### QAR kicks AHA back if:

| Issue | Problem |
|-------|---------|
| Missing associated tasks | Incomplete scope |
| Hazards not identified | Incomplete hazard ID |
| No RAC score | Can't assess risk |
| Controls don't reduce RAC | Inadequate protection |
| Wrong format | Doesn't meet spec |
| Missing signatures | Not briefed |
| Competent person not listed | No supervision |

---

## QAR Review Checklist

```
AHA REVIEW CHECKLIST:

[ ] Activity name clearly stated
[ ] Location specific
[ ] All associated tasks listed
[ ] Each hazard identified with RAC
[ ] Controls for EACH hazard
[ ] Controls will reduce RAC (III or lower)
[ ] PPE specified and adequate
[ ] Emergency procedures included
[ ] Competent person named
[ ] Workers sign-in sheet attached
[ ] AHA submitted before work starts

RESULT: [ ] APPROVED  [ ] KICK BACK
```

## Example Kick Back

**Subject:** AHA Kick Back - Electrical Rough-In

**Issue:** AHA incomplete. Missing associated tasks:
- Conduit bending not listed
- Underground routing (before slab) not addressed
- Working at elevation (scaffold) not covered

Also, RAC for "working at heights" rated II but only administrative controls (tethering) listed. Need engineering controls (guardrails) or additional PPE.

**Action:** Please revise AHA to include all associated tasks and add additional fall protection controls. Resubmit to RMS for re-review.
