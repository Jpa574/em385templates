# Fire Protection Inspection Procedures (DFOW)

## Fire Protection Defined Feature of Work

### Phase 1: Pre-Con (Before Installation)

**Contractor Must Complete:**
- [ ] Fire protection submittals approved (pipe, fittings, sprinklers, valves)
- [ ] Sprinkler heads correct type (pendent, upright, sidewall)
- [ ] Sprinkler head temperature rating per location
- [ ] Alarm valves approved
- [ ] Fire department connection type correct
- [ ] Pipe material per spec
- [ ] Coordination with ceiling layout
- [ ] AHA for fire protection approved
- [ ] Materials on site verified

**QAR Verification:**
- [ ] Verify submittals approved
- [ ] Confirm sprinkler type per area
- [ ] Verify alarm valve type (wet/dry)
- [ ] Check FDC type per NFPA 1963

**Forms/Docs Required:**
- Fire Protection Submittal Log
- Sprinkler Head Schedule
- Fire Alarm Coordination

---

### Phase 2: Work in Progress (During Installation)

#### Underground Fire Main
**Contractor Must Complete:**
- [ ] Pipe material correct (cast iron, ductile iron)
- [ ] Joints per spec (mechanical, push-on)
- [ ] Thrust blocks at bends, tees, caps
- [ ] Depth of cover per spec
- [ ] Pipe coating intact

**QAR Verification:**
- [ ] Verify burial depth
- [ ] Check joint type
- [ ] Verify thrust blocks
- [ ] Witness hydrostatic test

**Testing (Underground):**
- Hydrostatic: 200 psi or 1.5x working pressure (whichever higher) for 2 hours

#### Aboveground Fire Suppression
**Contractor Must Complete:**
- [ ] Pipe material per spec (schedule, type)
- [ ] Hanger spacing per MSS SP-58 (wet: 10', dry: 8')
- [ ] Fittings correct type
- [ ] Sprinkler heads correct type, spacing
- [ ] Sprinkler orientation correct
- [ ] Alarm valve installed per manufacturer
- [ ] Waterflow devices installed
- [ ] Fire department connections

**QAR Verification:**
- [ ] Check hanger spacing
- [ ] Verify pipe schedule
- [ ] Confirm sprinkler spacing (per design density)
- [ ] Verify orientation (pendent/downward)
- [ ] Check alarm valve installation

**Testing (Aboveground):**
- Hydrostatic: 200 psi or 1.5x working pressure for 2 hours
- Flow test: Verify design density

---

### Phase 3: Closeout (Final Inspection)

**Contractor Must Complete:**
- [ ] All hydrostatic tests passed
- [ ] Flow test performed (design density verified)
- [ ] Fire alarm connected
- [ ] All sprinkler heads installed (no missing)
- [ ] Cabinet/hoses if required
- [ ] O&M manuals (2 copies)
- [ ] As-builts marked up
- [ ] Fire department inspection passed

**QAR Verification:**
- [ ] Review test reports
- [ ] Verify flow test results
- [ ] Confirm all heads installed
- [ ] Check alarm notification
- [ ] Verify FDC accessibility

**Forms/Docs Required:**
- Underground Hydrostatic Test Report
- Aboveground Hydrostatic Test Report
- Flow Test Report
- Sprinkler Head Inventory
- Fire Protection Final Sign-off

## Common Fire Protection Deficiencies

| Deficiency | Spec Reference | Common Cause |
|------------|----------------|--------------|
| Missing thrust blocks | UFGS 21 13 00 | Joint failure |
| Wrong sprinkler type | Spec | Wrong temp rating |
| Hanger spacing too far | MSS SP-58 | Pipe failure |
| No drip legs | Spec | Low point accumulation |
| Wrong FDC thread | NFPA 1963 | FD can't connect |
| Alarm valve not wired | Spec | No notification |
| Pipe not supported | Spec | Sagging, damage |
| Missing fire stopping | Spec | Penetrations not sealed |
