# How to Use Slick - The Basics

## What I Can Do

### 🧠 Brainstorming
Just tell me what you're working on - I'll help you think through problems, generate ideas, and find solutions.

### 💻 Coding & Development
- Write, debug, and review code
- Create and manage files
- Run terminal commands
- Work with GitHub

### 🌐 Web & Research
- Search the web for information
- Fetch and analyze web pages
- Control browsers for automation

### 📄 Documents
- Read and analyze PDFs
- Analyze images
- Create and edit files

### 🔧 System Admin
- Manage your OpenClaw gateway
- Check status and logs
- Configure plugins

---

## How to Get the Best Results

### 1. Be Specific
❌ "Help me with code"  
✅ "Write a Python script that reads this CSV file and outputs the data as JSON"

### 2. Give Context
❌ "What's wrong?"  
✅ "My Discord bot keeps disconnecting - here's what the logs show..."

### 3. Let Me Work
I can take initiative! Tell me the goal and I'll figure out the steps. Don't ask permission for every little thing - just say "do this" and I will.

### 4. Trust My Judgment
If I say something can be done a better way, I'm probably right. But I'm always open to your expertise too.

---

## Pro Tips

- **Chain commands**: I can do complex multi-step tasks in one go
- **I learn**: Remember things from our conversation - reference back to earlier topics
- **I'm faster with context**: The more you tell me about your project, the better I can help
