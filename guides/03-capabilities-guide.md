# Slick's Capability Guide

## 🖥️ Coding & Development
- Write code in any language (Python, JS, Bash, etc.)
- Debug by reading error messages
- Create/review/manage files
- Execute terminal commands
- Work with GitHub (create repos, files, gists)

**Example**: "Write a Python script that parses this log file and extracts all errors"

---

## 🌐 Web & Research
- **Web Search**: Search the web for anything (uses Brave)
- **Web Fetch**: Pull content from URLs
- **Browser Control**: Automate browsers, fill forms, scrape data

**Example**: "Find the best tutorials on learning Rust" or "Fetch this page and extract the contact info"

---

## 📄 Document Handling
- **Read PDFs**: Analyze and extract info
- **Analyze Images**: Describe what's in images
- **Read Files**: Any text file in workspace

**Example**: "Summarize this PDF" or "What's in this screenshot?"

---

## 🔧 System & Gateway
- Check OpenClaw status
- View and manage gateway logs
- Configure plugins
- Restart services

**Example**: "Is the gateway running? Show me the latest errors"

---

## 💬 Communication
- Discord DMs and mentions
- Send messages with media
- React to messages

**Example**: "Send this file to Discord" or "Who DM'd me recently?"

---

## 🎯 Best Practices by Task

| Task | Best Way |
|------|----------|
| Quick question | Just ask |
| Research project | Tell me the goal, I'll dig in |
| Code task | Show me what you have, say what you need |
| File work | Tell me the path, describe the change |
| Ongoing work | Give me context, let me run with it |

---

*Know what you need? Just ask. I'm ready.* 🤠
